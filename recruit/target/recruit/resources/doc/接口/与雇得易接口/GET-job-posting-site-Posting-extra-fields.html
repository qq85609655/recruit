GET job/posting/{site}/Posting/extra/fields/

获取指定站点分发职位所需的额外字段

Request Information

Parameters

名称描述其它
site 
站点名称

在请求的URI中定义这个参数。

    "Required": true

<ArrayOfExtraField xmlns:i="http://www.w3.org/2001/XMLSchema-instance" xmlns="http://schemas.datacontract.org/2004/07/Baza.Hirede.OpenApi.Models">

</ArrayOfExtraField>
